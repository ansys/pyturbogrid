pyturbogrid_core#

Module in the ansys-turbogrid-api package that is internally used by modules in the pyturbogrid package for interactions with a running Ansys TurboGrid application.

Read a blade row from a *.tginit file.

Parameters#

pathstr

Name or path for the TGInit (*.tginit) file.

bladerowstr, default: ""

Optional blade row to be read, no entry corresponds to the first blade row.

secondarylist[str], default: []

A list of the secondary flow paths to include in the simulation.

autoregionsbool, default: True

Automatically generate opening regions in the passage inlet/outlet block to match the secondary flow path geometry.

includemeshbool, default: True

When set to true, the 3D secondary flow path meshes, along with their 2D boundaries, will be computed.

save_mesh(filename: str, onefile: bool = True, onedomain: bool = True, addsecondary: bool = True)#

Save generated mesh to a file.

Parameters#

filenamestr

Name of the mesh file to save.

onefilebool, default: True

If enabled (true), write all of the available meshes to a single mesh file.

onedomainbool, default: True

If enabled (true), combine any inlet and outlet domain meshes with the passage domain, to form a single assembly.

addsecondarybool, default: True

If enabled (true), any meshed secondary flow paths will be added to the mesh file.
